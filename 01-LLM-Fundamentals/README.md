# LLM Engineering Fundamentals

Building practical Large Language Model (LLM) applications with Python through API integration, prompt engineering, tokenization, web scraping, local models, and multi-step AI workflows.

---

## Overview

This repository contains a collection of projects that explore the core components of modern LLM applications. The projects demonstrate how language models interact with external data, how prompts influence model behavior, how local models can replace cloud APIs, and how multiple LLM calls can be combined into complete AI workflows.

---

## Projects

### Website Summarizer

A Python application that retrieves website content, removes unnecessary HTML elements, constructs prompts, and generates summaries using a locally hosted Llama 3.2 model through Ollama.

**Technologies**

* Python
* Requests
* BeautifulSoup
* Ollama
* OpenAI Python SDK

---

### AI Company Brochure Generator

A multi-step application that analyzes a company's website, identifies relevant pages using an LLM, gathers additional content, and generates a professional company brochure in Markdown.

**Technologies**

* Python
* OpenAI SDK
* BeautifulSoup
* JSON
* Markdown
