# Open-Source Generative AI with Hugging Face

## Overview

In this project, I explored how modern open-source large language models are built, loaded, and used for real-world AI applications. Rather than relying only on hosted APIs, I worked directly with models from the Hugging Face ecosystem to better understand how inference works behind the scenes.

Throughout this project, I learned how to run transformer models on GPU-enabled environments, use Hugging Face pipelines for rapid development, work directly with tokenizers and transformer models, and build an end-to-end AI application that converts meeting audio into structured summaries.

---

## What I Built

* Configured GPU-powered development environments using Google Colab
* Loaded and ran open-source transformer models from Hugging Face
* Built applications using the Hugging Face `pipeline()` API
* Worked directly with tokenizers to understand how text is converted into model inputs
* Explored transformer models beyond high-level abstractions
* Processed audio files and generated AI-powered meeting summaries
* Compared different open-source models and their capabilities
* Learned how model inference works from input text to generated output

## What I Learned

This project gave me a much deeper understanding of what happens inside an LLM after a prompt is submitted. Instead of treating language models as black boxes, I learned how text is tokenized, how transformer models process those tokens during inference, and how generated responses are produced one token at a time.

By working directly with Hugging Face models and building an AI-powered meeting summarization application, I gained practical experience developing AI systems using open-source tooling while strengthening my understanding of the core technologies that power modern Generative AI.
